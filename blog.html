What is DOM?
DOM (Document Object Model) represents the HTML page as a tree structure and allows JS to manipulate it.
How to select an element from the DOM?
Use getElementById(), getElementsByClassName(), querySelector(), or querySelectorAll() to select elements.
What is event delegation and why is it useful?
Event delegation is adding a listener on a parent to handle child elements. It saves memory and handles dynamic elements.
How to manipulate an element’s attributes and styles?
Use setAttribute() for attributes and style or classList to change styles.
What is the difference between innerHTML, innerText, and textContent?
innerHTML gets/sets HTML content, innerText gets/sets visible text, textContent gets/sets all text.
How to add or remove classes in DOM?
Use element.classList.add(), element.classList.remove(), or element.classList.toggle().
How to create a new element in DOM?
Use document.createElement("tagName") and append it with appendChild() or append().
How to remove an element from DOM?
Use element.remove() or parent.removeChild(child).
How to listen for events on elements?
Use element.addEventListener("event", callbackFunction).
What is parentNode, childNodes, and nextSibling?
They allow navigation in DOM tree: parentNode → parent, childNodes → children, nextSibling → next node.
